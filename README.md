# IQC-UI
在公司擔任資料工程實習生期間的專案，用來輸入進料檢驗紀錄資料的Web App，
<br>
以Python的Flask框架開發，並部署在Azure App Service的雲端服務上。

PS:僅附上此網頁實際使用畫面，並將公司業務相關資訊打碼。

<br>
